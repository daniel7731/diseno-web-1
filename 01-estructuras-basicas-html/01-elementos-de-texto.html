Énfasis y formato de texto

Existen varios elementos para dar formato al texto:

  • Da un énfasis fuerte (negrita). strong
  • Aplica énfasis (cursiva). i
  • Subrayado. usando propiedade de css
  • Tachado. usando propiedade de css
  • Underline. usando propiedade de css
